# SpaceNerd Privacy Policy

**Last updated:** 2026-03-04

SpaceNerd is a sensor-based sky pointing app that helps identify stars and planets.

## Data we do not collect
We do **not** collect, store, or sell personal data.
We do **not** require user accounts.
We do **not** use third-party analytics or advertising SDKs.

## Device permissions and how they are used
SpaceNerd may request the following permissions:

- **Camera**: used to show a live preview so you can point the phone at the sky.
- **Motion**: used to read device orientation so the app knows where you’re pointing.
- **Location (When In Use)**: used to compute which stars/planets are visible for your location.

These sensor values are used on-device for app functionality.

## Network access
SpaceNerd may access the network to fetch planet ephemeris data from NASA/JPL Horizons. Requests contain only the parameters required to obtain planet positions (e.g., time/epoch). No user account data is sent.

## Files stored on your device
SpaceNerd may save:
- planet ephemeris cache (e.g., `planetCoords.json`)
- optional debug logs (when enabled)

These files remain on your device unless you delete the app.
